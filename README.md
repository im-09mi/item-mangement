# 商品管理システム

## 概要
このシステムでは家電量販店の商品を管理することができます。
商品の新規登録や編集、削除が行えます。また検索機能も充実しているので商品検索が容易に行えます。

## 主な機能
- ログイン、ログアウト
- 会員登録
- ホーム画面（新着商品表示）
- 商品一覧画面（検索機能）
- 商品詳細画面
- 商品情報編集、削除
- 商品登録
- ユーザー情報画面
- ユーザー情報変更、削除

## 開発環境
```
PHP 7.4
MySQL 10.4
Laravel 8.83
```
